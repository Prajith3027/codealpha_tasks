# Netflix Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset using Python. The objective is to understand the structure of the dataset, identify trends, analyze content distribution, and extract meaningful insights through data visualization.

The analysis includes content type distribution, ratings analysis, country-wise content analysis, yearly content additions, genre analysis, and data quality assessment.

---

## Dataset

**Dataset Name:** Netflix Titles Dataset

The dataset contains information about movies and TV shows available on Netflix, including:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre

## Analysis Performed

### 1. Dataset Exploration

* Examined dataset shape and structure
* Reviewed column names and data types
* Identified missing values

### 2. Content Type Analysis

* Compared Movies and TV Shows
* Visualized content distribution

### 3. Rating Analysis

* Analyzed the most common content ratings
* Identified audience-targeted content categories

### 4. Country Analysis

* Examined content production by country
* Identified top contributing countries

### 5. Yearly Content Addition Analysis

* Analyzed trends in Netflix content additions over time
* Visualized growth in content availability

### 6. Genre Analysis

* Identified the most common genres
* Explored content diversity across categories

---

## Key Insights

* Movies significantly outnumber TV Shows on Netflix.
* TV-MA and TV-14 are among the most common content ratings.
* The United States contributes the highest amount of content.
* Netflix experienced rapid growth in content additions after 2015.
* International Movies, Dramas, and Comedies are among the most popular genres.

---

## Conclusion

The Exploratory Data Analysis reveals important trends within Netflix's content library. The platform offers a diverse range of content with a strong focus on movies and international entertainment. The increasing number of content additions over recent years highlights Netflix's expansion and growing global presence.
