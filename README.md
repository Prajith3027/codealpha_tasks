### 3. Sentiment Analysis on Amazon Product Reviews

#### Description

Applied Natural Language Processing (NLP) techniques to analyze Amazon customer reviews and classify them into Positive, Neutral, and Negative sentiments.

#### Project Workflow

* Data loading and preprocessing
* Review text cleaning
* Sentiment classification using TextBlob
* Sentiment distribution analysis
* Word cloud generation
* Insight extraction and reporting

#### Results Summary

| Sentiment | Review Count |
| --------- | -----------: |
| Positive  |       31,162 |
| Neutral   |        2,000 |
| Negative  |        1,497 |

#### Key Findings

* The majority of customer reviews expressed positive sentiment.
* Negative feedback represented a relatively small portion of the dataset.
* Customer satisfaction levels were generally high across reviewed products.

#### Tools & Libraries

* Python
* Pandas
* TextBlob
* WordCloud
* Matplotlib
* Seaborn
